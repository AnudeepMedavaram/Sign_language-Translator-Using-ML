# Sign Language Translator Using Machine Learning

## Overview
This project translates American Sign Language (ASL) hand gestures into text in real-time using computer vision and machine learning.

## Features
- Real-time hand tracking
- ASL alphabet recognition
- Sentence formation
- Text-to-speech output
- Web interface using Flask

## Tech Stack
- Python
- Flask
- OpenCV
- MediaPipe
- Scikit-learn
- NumPy

## How to Run
```bash
pip install -r requirements.txt
python app.py
